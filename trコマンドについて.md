## `tr`コマンド

文字の変換や削除を行うコマンド
#### コマンド
```
tr [オプション] SET1 [SET2]
```

#### 具体例
```
cat data.txt | tr "A-Z" "a-z"
```

##### 詳細
`tr`コマンドの中では

######  **文字の範囲**
`"A-Z"` や`"a-z"`といった記号は、文字数の範囲を表す。
- `A-Z`は、
	`ABCDEFGHIJKLNMOPQRSTUVWXYZ`
	と同じ意味になる。
同様に、
- `a-z`は、
	`abcdefghijklnmopqrstuvwxyz`
	と同じ意味になる。

すると、以下のように先頭から1文字ずつ順番に対応付けがなされ、置換される。
```
ABCD...XYZ
↓↓↓↓   ↓↓↓
abcd...xyz
```
###### **二つの集合の文字数が異なるときの挙動**
文字集合1 長さ =25 
`ABC....XYZ`

文字集合2 長さ =24
`ABC....xy` 

このとき、以下のようにマッピングされるようだ。
```
A → a
B → b
...
Y → y
Z → y
```
