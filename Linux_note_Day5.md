## プロセスについて

**プロセス**
	実行中のプログラムのこと。
	💡一つのコマンドにつき一つのプロセスが作られる。

#### プロセスを表示する
```markdown
ps [オプション]
```

#### 具体例
![[Pasted image 20260708164258.png]]

- **PID**
	プロセスのID
- **CMD**
	実行中のコマンド

#### オプション
`aux`
	全てのユーザーのプロセスを、詳細情報を合わせて表示
```
ps aux
```
💡オプションだが、ハイフンをつけなくていい

`a`
	`ps`コマンドを実行したユーザー以外の全てのユーザーのプロセス(つまり自分以外)
`u`
	詳細情報を合わせて表示する
`x`
	`ps`コマンドを実行したユーザーのすべてのプロセス(つまり自分)

## プロセス・ジョブの終了

#### コマンド
- ジョブを終了する
```
kill [オプション] %<ジョブID>
```

- プロセスを終了する
```
kill [オプション] <プロセスID>
```

#### 具体例
![[Pasted image 20260708171429.png]]
💡フォアグラウンドのジョブは`Ctrl + c`で終了できる。

#### 補足
正確には、<mark>killコマンドはシグナルを送信するコマンド</mark>
```
kill -<シグナル名> <プロセスID>
```

デフォルトでは、`TERM`という、終了を指示するシグナルを送信している。
```
kill 4965

↑
同じ
↓

kill -TERM 4956
```

#### 時々使うシグナル
`SIGKILL`
	強制終了を行う。
	`TERM`シグナルを受け付けない異常なプロセスなどに用いる。
	⚠️最終手段⚠️
```
kill -SIGKILL <プロセスID>
```

または、
```
kill -9 <プロセスID>
```
