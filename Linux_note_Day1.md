# シェルについて

#### **シェル**
LinuxのCLIを提供するソフトウェア。
つまるところ、<mark>ユーザーとLinux間のインターフェース</mark>。

- `sh`
	もっとも古いシェル
	機能が少ない
- `bash`
	shを基本として機能を拡張したシェル。
	Linux環境でデフォルトのログインシェル
- `zsh`
	この中で一番多機能。
	
#### **ターミナル**
ターミナルは<mark>入出力の画面を提供</mark>するソフトウェア。
ターミナルとシェルはあくまで別物。
